Занятие 1.
   
HTML
  Элементы и атрибуты.   Примеры:
  <p align="leftjustify" class="my-class">  text </p>
  <a href="google.com" target="_blank"> 
    - href="#idname" - строчный
  <img src="img"

ТЕГИ  
  р - абзац, h1-h6 - заголовки,
  a - ссылка, img - рисунок, 
  ul, ol, li - списки
  strong, em - выделенный текст
  table tr th td - таблица

  HTML - смысловая верстка. Поэтому предпочитаем <strong> <em> - <b>, <i>
  
CSS 
  <style> </style>
  <link rel="stylesheet" type="text/css" href="les2styles.css"/>
    селекторы
	шрифты цвет цвет фона text-decoration
	text-align: line-through, underline
	text-transform uppercase
	font-style: normal | italic | oblique
	
Задание на ПЗ
1. Добавить абзац с любым текстом в подразделе "Основы". Для этого нового раздела: 
	- определить шрифт Arial размер шрифта 15px;
	- задать цвет фона (светло-желтый), цвет текста (темно-синий)
	- изменить междустрочный интервал.

2. В разделе Серверные языки сделать все строки списка ссылками (ссылки должны вести на реальный ресурс)
Также изменить маркер списка (с диска на черточку) Можно ли сделать список нумрованым с помощью css?
3. Составить оглавление в виде нумерованного списка.
Каждый элемент списка должен быть гиперссылкой для перехода к соответствующему заголовку.
	<a href="#section2"> Второй раздел</a>
4. Добавить файл стиля style.css - подключить его к html-файлу
<link rel="stylesheet" type="text/css" href="style.css"/>
перенести в него стили
5. Добавить строчку к таблице (описание для языка JavaScript). 
Попробовать объединить в этой новой строчке 2ю и 3ю ячейки.
Для всей таблицы текст во второй колонке выровнять по ширене, а в третьей по правому краю. 
Также выделить текст во втором столбце немного увеличенной жирностью. 
	
ДЗ: Страничка с табличкой 3 х 4, составленной из рисунков 100px x 100px
Каждый рисунок должен быть ссылкой на веб-ресурс.
Все рисунки снабдить подписью, которая также должна быть ссылкой на данный ресурс.
